# Client-Server Authentication System

A TCP client-server authentication system written in C and Python.

## Features
- User registration
- Login authentication
- Session tokens
- Rate limiting
- Password hashing
- Concurrent clients using fork()
- Audit logging

## Files
- server_0919.c → Main C server
- client_0919.py → Python client
- Makefile → Build automation

make
